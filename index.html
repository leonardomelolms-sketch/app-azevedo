Acesso Bloqueado pelo Firebase

O banco de dados foi criado, mas as Regras de Segurança estão impedindo o salvamento dos dados. Você precisa liberar o acesso no painel do Firebase.

1. Vá em Firestore Database > Aba Regras

2. Cole este código e clique em Publicar:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
